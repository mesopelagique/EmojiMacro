# EmojiMacro

Macro file for emoji.

Some examples:
- Type @check_mark_button for `<:check_mark_button:>` ie. ✅
- Type @construction for `<:construction:>` ie. 🚧
- Type @lemon for `<:lemon:>` ie. 🍋
